While working with files, we cannot extract it from the req.body
    If we do, then it wont work

    if we use app.use(bodyParser.urlencoded({ extended: false })); this middleware
        urlencoded represent basically a text data, where file is a binary data

    So to make it work, we need to use different parser, and bodyParser doesnot include anything to handle this binary data
    So we need new package for that. which is "npm i multer"
        which handle both text and binary data
